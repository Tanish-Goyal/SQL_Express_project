# SQL_Express_project

To make this project work properly, you just need to firstly install some modules in your package:

1. Initialise npm in your project 
  Command: npm init -y
2. Install Express in your project
  Command: npm install express
3. Install body-parser in your project
  Command: npm install body-parser
4. Install MySql in your project
  Command: npm install sql
  
Finally run this project by open index file: node index.js.
This would open home page of the registration process.
